Как получить бесплатный ключ ElevenLabs:
  1. Перейдите на сайт ElevenLabs и зарегистрируйтесь (можно через Google-аккаунт).
  2. Кликните на иконку вашего профиля в левом нижнем углу экрана.
  3. Выберите пункт «Profile Settings».
  4. Найдите строку «API Key», нажмите на значок «глаза» для отображения и скопируйте его.
  5. Вставьте ключ в поле выше. Можно добавить несколько ключей (по одному на строку) — программа будет автоматически переключаться на следующий при достижении лимитов!

Как получить ключ Gemini бесплатно:
  1. Перейдите на сайт Google AI Studio.
  2. Войдите под своим Google-аккаунтом (если вы в РФ, используйте VPN).
  3. Нажмите кнопку «Get API key» в левом верхнем углу.
  4. Нажмите «Create API key», выберите проект и скопируйте созданный ключ.
  5. Вставьте полученный ключ в поле выше. Можно добавить несколько ключей (каждый с новой строки) — при исчерпании лимитов или ошибках программа автоматически переключится на следующий!
Как получить ключ Groq бесплатно:
  1. Перейдите на сайт Groq Console.
  2. Зарегистрируйтесь (для РФ может потребоваться VPN).
  3. Перейдите в раздел «API Keys» в боковом меню слева.
  4. Нажмите «Create API Key», укажите имя ключа и скопируйте его.
  5. Вставьте ключ в поле выше. Точно так же можно ввести несколько ключей по одному на строку.
