# MARKETING & SEO MANDATE
## Для модуля "SEO-Маркетолог" (Agent-Marketer)

Данный мандат описывает жесткие правила работы с отзывами Яндекс.Карты, 2ГИС и генерацией сообщений для пациентов.

[ПРАВИЛО: НИКАКОГО ВЕБ-СКРАПИНГА]
- Запрещено писать ботов для автоматического сбора отзывов с Яндекса через BeautifulSoup/Selenium, так как это ведет к блокировке по IP и капчам.
- Система должна использовать **полуавтоматический подход**: администратор копирует текст отзыва в локальную CRM, а нейросеть мгновенно генерирует ответ.

[ПРАВИЛО: ИНЪЕКЦИЯ LSI-КЛЮЧЕЙ]
- Любой сгенерированный ответ на позитивный или нейтральный отзыв **ОБЯЗАТЕЛЬНО** должен содержать 1-2 SEO-ключа для органического продвижения на картах.
- Скрипт `seo_agent.py` должен содержать жестко заданный список ключей (например: "лечение кариеса", "безболезненное удаление", "стоматология Самара").

[ПРАВИЛО: РАБОТА С НЕГАТИВОМ]
- Ответы на негативные отзывы **никогда не должны содержать оправданий или споров**.
- Формула ответа: `[Извинение] -> [Признание важности обратной связи] -> [Призыв связаться с главврачом по телефону X для решения проблемы]`.
- SEO-ключи в негативе использовать по минимуму, чтобы не выглядело как издевательство.

[ПРАВИЛО: ИНТЕГРАЦИЯ]
- Весь функционал Маркетолога должен быть доступен в едином интерфейсе `clinic_admin/templates/dashboard.html` (вкладка "Отзывы" или "Маркетинг").
- Запрещено создавать отдельные окна консоли или заставлять администратора запускать скрипты руками. Всё работает через HTTP POST к `main.py`.
